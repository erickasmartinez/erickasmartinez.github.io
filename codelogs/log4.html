Code log 4: CSS Color

  1. How does CSS accept color as a value?

    CSS accepts color as a value by giving color (s) to a CSS style property.

  2. What do these CSS properties do?
    • Color: This CSS property gives text color to a specific element component
    • Background-color: This CSS property gives a specific background color to a content section of an element
    • Border-color: This CSS property gives a color to the border around a specific element
  3. Examples of colors in these styles:
    • Named: A named color is a specific color for CSS, such as common colors, or color names you can think of that appear instantly on CSS because of its existence in the system when you start typing them.
    • Hexadecimal: A hexadecimal color is a color that starts with a # followed by a six numbered character that describes how much of a color is in a CSS property based on the colors of red (RR), green (GG) and blue (BB), with the value ranges from 00 to FF. The first two characters represent red, the next two green and the following blue, and the values 00 means that there is no color and FF means there is the highest amount of that color.
    • RGB: An RBG is a color specified by a function instead of characters. The function is rbg ( ), and inside the parenthesis are the same color values of red, green and blue. There are commas to distinguish between the three and to define the intensity of these three colors based on the integers of 0 to 255 or as a percentage value from 1% to 100%.
    • RGBA: RGBA is a color value that is similar to RBG but has an extra value called Alpha. This value determines how transparent the color is. The function is rgba ( ) and the value of Alpha ranges from 0, which means zero transparency to 1.0 meaning complete transparency.
