Code Log 2:CSS Measuring Units

  1. What is the difference between absolute and relative CSS length units?

    2. The difference between the two is that absolute CSS length units are fixed values that will remain the same even if there are changes in the parent or other structural elements. On the other hand, relative CSS length units are values that change in relation to other elements in the content.
